# Meetify

Meetify is a modern and lightweight meet-up management app built with Flutter.
It features a clean UI, smooth navigation, Firebase-powered authentication, and both Light & Dark theme support.
Designed for simplicity and speed, Meetify helps users manage meetings effortlessly with a beautiful and responsive interface.
